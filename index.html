Merging PDF Files

Make sure Ghostscript has been installed on the machine. Run 'gs -v' in a console to check this. gs can then be used to merge PDF files.

Here is an example. I have five PDF files in the current folder: a.pdf, b.pdf, c.pdf, d.pdf, and e.pdf. They can be merged into one PDF file f.pdf in the same folder by the following command. 

$ gs -dNOPAUSE -sDEVICE=pdfwrite -sOUTPUTFILE=./f.pdf -dBATCH a.pdf b.pdf c.pdf d.pdf e.pdf
